== Description ==
Somos uma plataforma de rastreamento de tráfego digital, voltada para lançadores de produtos digitais que têm como principal objetivo entender melhor seus dados de lançamento. Tudo isso em um único lugar.

A Lead Tracker consegue rastrear toda a jornada do seu lead, desde a captação até a compra. Com esses dados, nossos usuários conseguem entender seus melhores canais de conversão, assim como campanhas e até criativos, para assim tomarem decisões ainda mais assertivas.

Nosso plugin do Wordpress vai lhe ajudar a instalar nosso pixel de rastreio na sua página de captação em apenas alguns cliques, ideal para quem não entende de programação. Com ele, você conseguirá dar o primeiro passo par ater um rastreamento de leads e de vendas preciso, de forma rápida e prática.

== Installation ==

It's super easy to install LeadTracker's plugin and get it working.

1. Upload `leadtracker.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter your user and select your Account in the plugin settings screen
